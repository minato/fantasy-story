# 世界観設定

このディレクトリには、ファンタジー世界の設定に関する情報が格納されています。

## 含まれる内容

- 地理的特徴（大陸、国、都市、特殊な場所など）
- 歴史的背景（重要な出来事、時代区分など）
- 魔法システム（魔法の仕組み、種類、制限など）
- 種族（人間以外の知的生命体、その特徴など）
- 文化と社会（言語、宗教、政治体制など）
- 技術レベル（武器、交通手段、日常生活など）

## ファイル命名規則

- 明確なカテゴリごとにファイルを分ける
- ファイル名は内容を端的に表す英語の単語を使用
- 複数の単語を使う場合はスネークケース（例: `magic_system.md`）を使用
